# Web Design Practical No. 20 - Interactive Quiz

An interactive, web-based quiz application developed as part of the Web Design curriculum.
This project demonstrates the use of HTML, CSS, and JavaScript to create a functional user interface with real-time feedback.

## 🚀 Live Demo
You can view the live project here:
[https://gyash28.github.io/WD_practical_no_20/](https://gyash28.github.io/WD_practical_no_20/webdesign_quiz.html)

## 🛠️ Features
- **Responsive Design:** Works on both desktop and mobile browsers.
- **Interactive UI:** Dynamic questions and options.
- **Instant Scoring:** Provides immediate results upon completion.
- **Clean Code:** Structured specifically for academic practical requirements.

## 💻 Technologies Used
* **HTML5:** For semantic page structure.
* **CSS3:** For styling and layout.
* **JavaScript:** For quiz logic and interactivity.
