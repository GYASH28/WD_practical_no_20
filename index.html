Unit 1 – Introduction to Web Designing

1.1 What is HTML?

The first version of HTML was written by Tim Berners-Lee in 1993. HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page using a series of elements.

1.2 Basic Web Terms

  • Web Page: A document displayed in a web browser, written in HTML.
  • Website: A collection of web pages grouped together with links connecting them.
  • Web Server: A computer that hosts a website on the Internet.
  • Search Engine: A web service that helps you find other web pages (Google, Bing, Yahoo).

1.3 Structure of an HTML Page

Tag Purpose Type
<!DOCTYPE html> Specifies HTML5 version Unpaired
<html> Root element, wraps all HTML content Paired
<head> Contains title, CSS/JS links Paired
<title> Title shown in browser tab — placed inside <head> Paired
<body> All visible content goes here Paired

1.4 Heading Tags

HTML provides 6 levels of headings: <h1> to <h6>. <h1> is the largest/most important, <h6> is the smallest. All are paired tags.

1.5 Other Block-Level Tags

  • <p>: Paragraph tag — creates paragraphs with spacing.
  • <pre>: Preformatted text — preserves spaces and line breaks. Fixed-width font.
  • <br>: Line break — unpaired tag, no closing tag.
  • <hr>: Horizontal rule — draws a horizontal line.
  • <center>: Centers text (not supported in HTML5, use CSS instead).
  • <div>: Block-level container element.
  • <span>: Inline container element.

1.6 Formatting Tags

  • <b>: Physical bold (just visual)
  • <strong>: Logical bold — implies high importance (used for SEO)
  • <i>: Physical italic
  • <em>: Logical italic — emphasis
  • <mark>: Highlights text with yellow background
  • <img>: Image tag — unpaired tag

Unit 2 – Lists and Linking HTML Documents

2.1 Types of Lists

Type Tag Description
Ordered List <ol> Numbered list (1, I, i, A, a)
Unordered List <ul> Bulleted list (disc, circle, square, none)
Description List <dl> Term-definition format like a dictionary
List Item <li> Used inside <ol> and <ul>
Data Term <dt> Used inside <dl> for the term
Data Definition <dd> Used inside <dl> for the description

2.2 Anchor Tag (Linking)

  • The anchor tag <a href="..."> creates hyperlinks to other pages or URLs.
  • href = Hypertext Reference — points to destination page.
  • target="_blank" opens link in a new tab.
  • target="_self" opens in the same tab (default).
  • target="_parent" opens in parent frame.

2.3 Link States / Appearance

  • Unvisited link: Underlined and Blue
  • Visited link: Underlined and Purple
  • Active link: Underlined and Red

2.4 Marquee

  • Default marquee speed = 6px per second
  • scrollamount attribute changes speed
  • scrolldelay adds delay between scrolls
  • Stop on mouseover: onmouseover="stop()"
  • Start on mouseout: onmouseout="start()"

2.5 Special Characters

  • &nbsp; = Non-breaking space
  • &ensp; = En space
  • &emsp; = Em space (wider)
  • <nav> tag groups navigation links

3.3 CSS Background Properties

  • background-color: Sets background color
  • background-image: Sets an image as background
  • background-repeat: repeat-x: Repeats image horizontally
  • background-repeat: repeat-y: Repeats image vertically
  • background-repeat: no-repeat: Shows image once only
  • background-attachment: fixed: Image stays fixed while scrolling
  • background-attachment: scroll: Image scrolls with the page
  • background-position: Sets position of background image

3.4 CSS Border Styles

Values for border-style: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden

groove = 3D grooved effect | ridge = 3D ridged effect | inset = 3D inset | outset = 3D outset

border-radius: 50% makes an image appear as a circle.

3.5 CSS Padding

Padding creates space around an element's content, inside any defined borders. Properties: padding-top, padding-right, padding-bottom, padding-left. Negative values are NOT allowed.

Unit 4 – Table, Frames and Forms

4.1 HTML Table Tags

Tag Purpose
<table> Defines the table
<tr> Defines a table row
<th> Defines a header cell (bold, centered by default)
<td> Defines a data cell
<caption> Table caption
<thead> Groups header rows
<tbody> Groups body rows
<tfoot> Groups footer rows

4.2 Table Attributes

  • border: Width of border around table
  • cellspacing: Space between cells
  • cellpadding: Space between cell wall and content
  • rowspan: Merges cells vertically (row-wise)
  • colspan: Merges cells horizontally (column-wise)
  • align: Aligns table to left, right, center
  • bgcolor: Background color of table/cell
  • valign: Vertical alignment — top, middle, bottom

4.3 HTML Forms

  • GET method: Appends form data to the URL (visible in address bar)
  • POST method: Sends form data in the request body (not visible in URL)
  • action attribute: Specifies where to send the form data
  • input type="text": Single line text input
  • input type="password": Obscured/hidden text input
  • input type="reset": Clears all form controls
  • input type="submit": Submits the form
  • <select> uses <option> tags for dropdown menu items
  • <textarea>: Multi-line text input

4.4 Frames / iFrame

<iframe> is used to embed another HTML document inside the current page. The src attribute specifies the URL of the embedded document.

Unit 5 – HTML Style (CSS)

5.1 CSS Basics

  • CSS = Cascading Style Sheet
  • Used to design and style HTML elements
  • Syntax: selector { property: value; } — called a Declaration Block
  • Three ways to insert CSS: External, Internal, Inline

5.2 CSS Box Model (inside to outside)

Content → Padding → Border → Margin

  • Content: Innermost — actual text/images
  • Padding: Space between content and border (transparent)
  • Border: Outline around padding
  • Margin: Outermost — space outside the border

5.3 Display, Position, Z-index

  • display: block — default for <div>
  • display: none — removes element from layout (space gone too)
  • visibility: hidden — hides element but keeps its space
  • position: fixed — element stays in place while page scrolls
  • position: absolute — positioned relative to parent
  • position: relative — positioned relative to itself
  • z-index — controls stacking order (overlap order)
  • margin: auto — centers a block element horizontally

5.4 CSS Animations

  • Animation steps are defined using @keyframes rule
  • animation-name: Name of the @keyframes
  • animation-duration: How long the animation runs
  • animation-delay: Delay before starting
  • animation-iteration-count: How many times to run

5.5 Margin vs Padding

  • Margin: Space outside the border — can be negative — auto is allowed
  • Padding: Space inside the border — cannot be negative
  • Both have: top, right, bottom, left variants

Unit 6 – Website Designing Tools

6.1 Photoshop Tools

  • Lasso Tool: Used for complex freehand selections
  • Magic Wand: Selects areas based on color similarity
  • Quick Select: Quickly selects a region
  • Clone Stamp: Copies/retouches pixels from one area to another
  • Healing Brush / Spot Healing: Fixes blemishes and spots

6.2 Adobe XD vs Procreate

Feature Adobe XD Procreate
Purpose UI/UX wireframing and prototyping Digital painting and illustration
Platform Desktop + mobile iPad/iPhone only
Target UI/UX Designers, developers Artists, illustrators

6.3 HTTP Protocol

  • HTTP = Hypertext Transfer Protocol
  • HTTP is a stateless protocol (each request is independent)
  • Uses client-server model — client requests, server responds
  • GET: Retrieves data from server
  • POST: Submits/creates data on server
  • PUT: Replaces an existing resource
  • DELETE: Deletes a resource
  • PATCH: Partially updates a resource

6.4 HTTP Status/Error Codes

Code Meaning
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
500 Internal Server Error
503 Service Unavailable
502 Bad Gateway

6.5 Web Hosting

  • Web hosting = renting/purchasing space on the web to host a website
  • Shared Hosting: Multiple websites share same server
  • VPS (Virtual Private Server): Dedicated virtual resources on shared hardware
  • Dedicated Hosting: Entire server dedicated to one user

6.6 DNS (Domain Name System)

  • DNS translates human-readable domain names into IP addresses
  • There are 13 different IP addresses that serve the DNS root zone
  • Hundreds of redundant root servers exist worldwide
